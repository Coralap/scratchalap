# Scratchalap
A ML model editor built with Electron, Blockly, and FastAPI.

## Setup Instructions
1. **Install JavaScript Dependencies**:
    ```bash
    npm install

2. **Install Python Dependencies:**:

    ```bash
    pip install -r requirements.txt

## Development
Run the frontend and FastAPI backend together:

    npm run dev

## Build
Generate a Windows exe:

    npm run build
